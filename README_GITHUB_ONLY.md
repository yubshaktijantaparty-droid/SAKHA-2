## If Something Doesn't Work

### API Key Issues
```
❌ Error: "Please configure your OpenRouter API key"
✅ Solution: Go to Settings and add your key
```

### Models Not Showing
```
❌ Dropdown is empty
✅ Solution: Make sure API key is saved in Settings
```

### No Response from AI
```
❌ Message sent but no response
✅ Solution: 
   - Check API key is correct
   - Wait 10-30 seconds (API processing)
   - Check internet connection
   - Verify OpenRouter is working
```

### Chat Not Saving
```
❌ Chat history lost after refresh
✅ Solution: Check browser allows localStorage
```
