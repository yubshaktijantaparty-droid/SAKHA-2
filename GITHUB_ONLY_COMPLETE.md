## 🎓 Learn More

### How It Works
1. You paste OpenRouter API key
2. App stores it in browser localStorage
3. When you send a message:
   - Browser sends request to OpenRouter
   - OpenRouter calls the AI model
   - Response comes back to your browser
   - App displays it to you
4. Chat history saved locally in browser

### Why This Is Better
- **Simpler**: No server setup needed
- **Cheaper**: Free hosting + pay-per-use API
- **Faster**: Direct API calls, no server overhead
- **Private**: Data never leaves your machine
- **Scalable**: No backend limits
